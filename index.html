Estruturas de Textos

Em HTML, a estrutura de texto refere-se à forma como o conteúdo textual é organizado e apresentado em uma página da web. Isso envolve o uso de diferentes elementos HTML para definir e estruturar o texto de maneira significativa e acessível.

1. h1, h2, h3, ..., h6: São usados para criar títulos e subtítulos, sendo o h1 o mais importante e o h6 o menos importante. Eles também ajudam na estruturação e acessibilidade do conteúdo.
2. p: Utilizado para criar parágrafos de texto.
3. ul e ol: ul é uma lista não ordenada, onde os itens são marcados com pontos; e ol é uma lista ordenada, onde os itens são numerados.
. li: Utilizado dentro de ul e ol para definir cada item da lista.
5. blockquote: Usado para citações longas ou blocos de texto destacados.
6. pre Mantém a formatação do texto exatamente como está no código-fonte, útil para exibir código ou texto formatado.
7. hr: Cria uma linha horizontal para dividir seções do texto.
8. br Utilizado para quebras de linha dentro de um parágrafo ou outro elemento.
Estes são apenas alguns dos elementos básicos para estruturar textos no HTML. A combinação desses elementos permite criar uma hierarquia clara e facilita a formatação e a apresentação do conteúdo em páginas da web.

Semanticas

Semântica em HTML refere-se à prática de atribuir significado e estrutura corretos aos elementos HTML para descrever adequadamente o conteúdo de uma página da web. Em outras palavras, trata-se de usar elementos HTML que expressem o significado e a função do conteúdo, em vez de apenas sua aparência visual.

Quando você utiliza a semântica correta em HTML, está fornecendo informações adicionais sobre o conteúdo da página, não apenas para os usuários que a visualizam, mas também para os mecanismos de busca e outros dispositivos que podem estar acessando a página (como leitores de tela para deficientes visuais). Por exemplo: Em vez de usar um div genérico para agrupar uma seção de conteúdo, pode-se usar elementos semânticos como header, nav, main, article, section e footer para expressar claramente a estrutura da página. Utilizar h1 - h6 para títulos e subtítulos em vez de apenas alterar o tamanho da fonte com CSS. Utilizar p para parágrafos, ul e ol para listas, blockquote para citações, entre outros, conforme apropriado. Ao usar semântica em HTML, você está não apenas melhorando a acessibilidade da sua página para usuários com diferentes necessidades e dispositivos, mas também contribuindo para uma melhor compreensão do conteúdo por parte dos motores de busca, o que pode influenciar positivamente a visibilidade e o ranking da sua página nos resultados de pesquisa.

Style

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para descrever a apresentação visual de um documento HTML ou XML. Em outras palavras, enquanto o HTML define a estrutura e o conteúdo de uma página da web, o CSS controla sua aparência e estilo.

Seletores: Os seletores CSS são usados para selecionar os elementos HTML aos quais as regras de estilo se aplicam. Eles podem ser baseados em elementos específicos, classes, IDs, atributos, etc. Propriedades e Valores: As propriedades CSS definem as características de estilo que você deseja aplicar aos elementos selecionados, como cor, tamanho da fonte, margens, preenchimento, etc. Cada propriedade tem um valor associado que determina como essa propriedade será aplicada. Classes e IDs: Classes e IDs são atributos que podem ser adicionados a elementos HTML para identificá-los de forma única ou agrupá-los para aplicar estilos comuns a vários elementos. Cascata: O termo "cascata" em CSS refere-se à maneira como as regras de estilo são aplicadas e podem ser substituídas. As regras definidas em arquivos de estilo externos, estilos embutidos e estilos definidos inline são aplicados de acordo com a hierarquia e a especificidade do seletor. Box Model: O Box Model é um conceito fundamental em CSS que define como o espaço é distribuído ao redor e dentro de um elemento HTML. Ele consiste em conteúdo, preenchimento, borda e margem. Media Queries: As Media Queries são usadas para aplicar estilos diferentes com base em características específicas do dispositivo, como largura da tela, tipo de dispositivo, resolução, etc., permitindo criar designs responsivos que se adaptam a diferentes tamanhos de tela. Transições e Animações: CSS também permite criar efeitos de transição e animação para elementos HTML, adicionando interatividade e dinamismo às páginas da web.
